# Website Resmi SDN Dawuan Tengah VI

Selamat datang di repository website resmi **SDN Dawuan Tengah VI**!

Website ini dibuat untuk memberikan informasi terbaru mengenai sekolah, termasuk berita, profil sekolah, kegiatan, dan fitur-fitur menarik lainnya.

## Fitur Utama

- **Halaman Beranda**: Sambutan dan informasi singkat sekolah.
- **Halaman Profil**: Menampilkan profil sekolah, visi dan misi, serta struktur organisasi.
- **Halaman Berita**: Berisi informasi kegiatan sekolah, termasuk Persami dan acara penting lainnya.
- **Desain Responsif**: Tampilan website sudah disesuaikan agar nyaman dilihat di HP dan komputer.
- **Animasi & Efek Transisi**: Website lebih interaktif dan menarik dilihat.
